# Support

Looking for help with SuperForm? Here's how to get support.

## 📖 Documentation

Before reaching out, check if your question is answered in our docs:

- [Quick Start Guide](docs/QUICK_START.md) — Get up and running in minutes
- [API Reference](docs/API.md) — Detailed API documentation
- [Examples](docs/EXAMPLES.md) — Real-world usage examples
- [FAQ](docs/FAQ.md) — Frequently asked questions
- [Troubleshooting](docs/FAQ.md#troubleshooting) — Common issues and solutions

## 💬 Getting Help

### GitHub Discussions

For questions, ideas, and community support, use [GitHub Discussions](https://github.com/SilentCoder-HI/superform/discussions):

- **Q&A** — Ask questions and get answers from the community
- **Ideas** — Share feature ideas and suggestions
- **Show & Tell** — Share what you've built with SuperForm

### GitHub Issues

For bug reports and feature requests, use [GitHub Issues](https://github.com/SilentCoder-HI/superform/issues):

- 🐛 [Report a Bug](https://github.com/SilentCoder-HI/superform/issues/new?template=bug_report.md)
- ✨ [Request a Feature](https://github.com/SilentCoder-HI/superform/issues/new?template=feature_request.md)

> **Note**: Please search existing issues before creating a new one to avoid duplicates.

## 🔒 Security Issues

If you discover a security vulnerability, please **DO NOT** open a public issue. Instead, follow our [Security Policy](SECURITY.md) for responsible disclosure.

## 📋 Before Asking for Help

To help us help you faster, please include:

1. **SuperForm version** (`npm list @silentcoderhi/superform`)
2. **Node.js version** (`node --version`)
3. **Framework** (React version, Next.js, etc.)
4. **Minimal reproduction** — a code snippet or repo that demonstrates the issue
5. **Expected vs. actual behavior**
